# Trabalho de Pesquisa — Algoritmos II

## Instruções
Criar uma aplicação que demonstre o uso das três pesquisas estudadas:
- Sequencial
- Indexada
- HashMap

## Regras
- Trabalho individual.
- Enviar apenas o link do repositório ou um arquivo para download que eu possa baixar e executar localmente.
- A tabela onde serão feitas as buscas deve estar populada com, no mínimo, 5.000 registros.
- As três pesquisas devem usar a mesma fonte/tabela de dados.
- Demonstrar o tempo que cada pesquisa leva para ser executada (incluir medições e ambiente de execução).

## Entregáveis
- Código-fonte executável ou link para repositório com README contendo instruções de execução.
- Script ou instruções para popular a tabela com ≥ 5.000 registros.
- Relatório com os tempos obtidos e descrição do ambiente de testes.

## Ideia Projeto
- Backend: Flask (Python)
- Frontend: HTML + Bootstrap ou Tailwind
- Banco: SQLite (já embutido)
- Biblioteca de medição de tempo: time (módulo padrão)
